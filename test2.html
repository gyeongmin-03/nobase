- 주요 공통 속성

- `readonly` : 읽기 전용

- `disabled` : 비활성화

- `autocomplete` : 자동 완성

- `autofocus` : 웹 페이지 로드시 초기에 포커싱 설정

- `placeholder` : 입력 폼에 희미하게 설명을 보여줌

- `required` : 필수 속성

- `spellcheck` : 오타 점검
